# 🎓 Student Performance Analysis Dashboard

> An interactive Power BI dashboard designed to analyze student academic performance, attendance, enrollment trends, and ranking insights using data visualization techniques.

---

## 📌 Project Overview

The **Student Performance Analysis Dashboard** provides a comprehensive analysis of student performance across multiple academic and demographic factors. It enables educational institutions to monitor student progress, identify top performers, analyze attendance trends, and gain actionable insights for data-driven decision-making. :contentReference[oaicite:0]{index=0}

---

## 🚀 Objectives

- Analyze overall student academic performance.
- Monitor attendance and its impact on marks.
- Identify top-performing students.
- Compare departmental performance.
- Analyze enrollment demographics.
- Provide interactive reports for stakeholders.
